# T-Master 官网架构与菜单规划方案 
# T-Master 공식 웹사이트 아키텍처 및 메뉴 기획안

为了清晰呈现 T-Master 涵盖“硬件 · 装备 · 平台”的全方位品牌形象，并在现阶段以单页(One-Page) 的形式提供流畅的用户体验，我们规划了以下网站层级与导航结构：

T-Master가 아우르는 "하드웨어·장비·플랫폼"의 종합적인 브랜드 이미지를 명확히 보여주고, 현 단계에서 원페이지(One-Page) 형식으로 매끄러운 사용자 경험을 제공하기 위해 다음과 같은 내비게이션 구조를 기획했습니다.

## 🗺️ 2. 页面区块规划 (Page Section Breakdown / 페이지 섹션 구성)

整个网站采用向下滚动浏览的单页面架构，分为以下四大核心区块：
전체 웹사이트는 아래로 스크롤하여 탐색하는 원페이지(One-Page) 구조로 설계되며, 다음과 같은 4개의 핵심 섹션으로 나뉩니다.

### 📍 [Section 1] 首页 (Hero Section / 메인 비주얼 섹션)
*   **视觉呈现 (Visual / 비주얼)**: 全屏展示高端高尔夫生活方式的高清视频/大图 (类似 PXG 风格)。 / PXG 스타일과 유사한 프리미엄 골프 라이프스타일을 담은 고화질 풀스크린 영상 또는 이미지.
*   **核心传递 (Message / 핵심 메시지)**: "T-Master: 全方位高尔夫解决方案 (포괄적인 골프 솔루션)"
*   **呼吁按钮 (CTA / 콜투액션)**: “探索更多 / 자세히 보기” 按钮，引导用户向下滚动。

### 📍 [Section 2] 产品中心 (Product Lineup / 제품 라인업 섹션)
产品中心由三个子模块组成，凸显品牌从硬件制造到周边装备的专业性：
브랜드의 하드웨어 제조 기술력부터 주변 장비까지의 전문성을 돋보이게 하는 3개의 하위 모듈로 구성됩니다.

1.  **智能硬件 (Smart Systems / 스마트 하드웨어)**
    *   自动发球机 (Auto Tee-up / 오토 티업)
    *   推杆练习器 (Putter Trainer / 퍼터 트레이너)
    *   自动配球机 (Ball Distributor / 자동 볼 배분기)
    *   模拟器 (Simulator / 시뮬레이터)
2.  **专业装备 (Premium Gear / 전문 장비)**
    *   展示 T-Master 追求极致性能的球杆与相关配件。 / T-Master의 완벽한 퍼포먼스를 추구하는 골프 클럽 및 액세서리 전시.
3.  **生产实力 (Global Manufacturing / 글로벌 생산 능력)**
    *   介绍中国本土强大的生产基地及 ODM/OEM 业务。 / 중국의 강력한 생산 기지 및 글로벌 ODM/OEM 비즈니스 역량 소개.

### 📍 [Section 3] 数字化生态 (Digital Platform / 디지털 플랫폼 섹션)
介绍即将上线的应用程序，强调软硬件结合的未来生态：
곧 출시될 애플리케이션을 소개하며, 소프트웨어와 하드웨어 결합의 미래 생태계를 강조합니다.

*   **T-Master APP 核心功能预告 (APP 주요 기능 프리뷰)**
    1.  **智能连接 (스마트 연결)**: 实现手机与所有硬件的无缝连接及数据同步。
    2.  **社交竞技 (소셜 및 경쟁)**: 提供线上约战与成长路径分享。
    3.  **专业赋能 (전문성 강화)**: 基于 AI 的挥杆分析与定制训练计划。

### 📍 [Section 4] 联系我们与尾部 (Contact & Footer / 문의하기 및 하단 섹션)
*   **B2B 合作表单 (B2B 협력 문의 양식)**: 收集潜在客户的姓名、邮箱及合作意向 (이름, 이메일, 협력 의향 수집).
*   **全球业务布局 (글로벌 비즈니스)**: 标注中、韩双核驱动的跨国业务格局 (한국·중국 중심의 글로벌 비즈니스 인프라 표기).
*   **法律指引 (Footer / 하단 정보)**: 包含版权声明 (Copyright)、隐私政策 (개인정보처리방침) 及服务条款 (이용약관).
